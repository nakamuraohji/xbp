マイコンボード

マイコンボードとは、マイクロコンピュータと入出力回路などの周辺回路を1枚の基板に乗せ、手軽にマイコンを利用するための回路。
機能を拡張するインプットデバイスとアウトプットデバイスがある。

Arduino

Arduinoとは、入出力ポートがあるマイコンボードのこと。
プログラムを用いることで主に電子工作と言われる様々な動作を簡単に行うことが出来る。

授業で使用したArduinoのプログラム

  • void setup() { } 最初に一回だけ実行されるプログラム

  • void loop() { } ずっと繰り返し実行されるプログラム

  • pinMode 指定したピンを入力用,出力用を設定する

  • digitalWrite(HIGH)/digitalWrite(LOW) LEDの点灯、消滅に用いる

  • delay 指定時間まで待機(単位はms)

For文

繰り返しの動作を行う。iは変数

If文

if/elseにより条件分岐の動作を行う。
